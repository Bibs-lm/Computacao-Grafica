## 📚 Sobre o Projeto

Este projeto foi desenvolvido como trabalho prático da disciplina de **Computação Gráfica** da **PUC Minas**.
O objetivo é implementar algoritmos fundamentais da área, com visualização em uma **matriz de pixels (área de desenho)**, permitindo interação direta via mouse.

---

## 🎯 Objetivos

* Implementar algoritmos de **rasterização**
* Aplicar **transformações geométricas 2D**
* Realizar **recorte de retas**
* Trabalhar com uma **interface gráfica interativa**

---

## 🧱 Funcionalidades

### ✏️ Rasterização

* Desenho de retas utilizando:

  * Algoritmo DDA
  * Algoritmo de Bresenham
* Desenho de circunferências:

  * Algoritmo de Bresenham

---

### 🔄 Transformações Geométricas 2D

* Translação
* Rotação
* Escala
* Reflexão:

  * Em relação ao eixo X
  * Em relação ao eixo Y
  * Em relação à origem (XY)

⚠️ Parâmetros definidos pelo usuário

---

### ✂️ Recorte de Retas

* Cohen-Sutherland (regiões codificadas)
* Liang-Barsky (equação paramétrica)

---

### 🧠 Estruturas

O sistema trabalha com:

* Pontos
* Retas
* Circuferências

---

## 🖱 Interação

* Interface baseada em cliques (mouse)
* Área de desenho representando uma matriz de pixels
* Seleção e manipulação direta dos objetos gráficos
